UI Technical Menu

Quicker access to Technical Menu items

Save clicks and scrolling!

This module explodes the Technical menu and places it at the top level of the interface. This allows you to access all technical menu items without having to scroll to the bottom of the Settings menu and without needing to expand the groups to get to individual menu items
